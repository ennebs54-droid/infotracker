# Delivery Tracking Website

A static delivery tracking website hosted on Netlify.

## Files

- `index.html` — Main tracking page
- `style.css` — Styles
- `script.js` — Tracking logic

## Deploy to Netlify

1. Push this folder to a GitHub repository
2. Go to [netlify.com](https://netlify.com) → **Add new site** → **Import from Git**
3. Select your repo — Netlify auto-detects `netlify.toml`
4. Click **Deploy site**

## Valid Tracking Number

The only accepted tracking number is: **TRKBU372**
